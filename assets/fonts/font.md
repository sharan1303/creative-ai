Place a free TTF/OTF font here for text overlays.

Recommended: Google Noto Sans (SIL Open Font License)

- Download: <https://fonts.google.com/noto/specimen/Noto+Sans>
- File to use: NotoSans-Regular.ttf (or your preferred weight)

The app auto-detects the first .ttf/.otf/.ttc file in this folder.
No configuration needed.

Docker compose already mounts ./assets to /app/assets in all services.
